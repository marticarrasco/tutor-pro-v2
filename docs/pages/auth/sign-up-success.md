# Sign Up Success (`/auth/sign-up-success`)

## Purpose
Confirms account creation and instructs the user to verify their email before accessing the dashboard.

## Content & interactions
- Displays a success icon with supportive copy emphasising that a confirmation link has been emailed.
- Offers a single primary action button that returns the user to the login page once they have completed verification.
- Reuses the authentication card layout and logo for consistent onboarding branding.
